OnlineShop - Nabawi Herbal | nabawiherbal.com
Nabawi Herbal

is a company that specializes in the distribution of herbal medicines and is the largest pioneer herbal distribution center in Indonesia. The company has several branch outlets and partner agents throughout Indonesia.


As a former employee of PT Nabawi Herbal, I have created an official website for the company using the Laravel framework in MVC format and adding several public APIs. I used Laravel 8, Javascript, Bootstrap 5, and several other libraries in developing this website.


The purpose of this official website is to provide information and facilitate transactions in one website. Every user who wishes to make a transaction is required to log in, as the website routes are restricted based on authentication and user roles. The login and registration form use Google Recaptcha V.2. After successful registration, users must verify with an OTP code sent to the registered email, otherwise the transaction process cannot be performed. After successful verification, users can log in and perform transactions.


Users can select the desired products and add them to the shopping cart, then fill in the payment form. There is a public API, Raja Ongkir, which calculates the shipping costs of goods to the destination address that has been filled in the payment form. After completion, users will be directed to their order list and asked to make payment. Payment is made through the Midtrans payment gateway, so there are several payment options offered. The website also uses a webhook from Midtrans to the website to automatically update the payment status, so manual checking is not necessary. After successful payment, users can wait for the order to be processed by the admin and can also upload transaction proofs as another option.


This website also implements SEO to facilitate the search for information on this website and is integrated with Google Analytics. Although no intensive optimization was performed, the results from Google Analytics have a good impact for adding features and regularly maintaining the website. The website is responsive, making it easy to access via mobile devices since many users access the website through mobile devices.

Tags :
Laravel, PHP, HTML, Css, SEO, Bootsrap, Javascript, API, Responsive, Midtrans, Raja Ongkir, Payment Gateway, OTP, Google Recaptcha, Google Search Console, Google Analytics.
SISFO - SI : Institut Teknologi Dan Bisnis Ahmad Dahlan | sisteminformasiitbad.info
SI : ITB AD

is one of the study programs at the Ahmad Dahlan Institute of Technology and Business campus. The head of the SI: ITB AD study program asked me directly to create a system for submitting final project and KKL titles for SI: ITB AD students. I created a website using Laravel 8, Javascript, Bootstrap, and several other libraries. The goal of creating this website is to facilitate students and staff in submitting final project and KKL titles in the future.


The flow of this website starts with registering an email and student number. After that, students will be asked to verify their account via email. If students do not verify their email, the menu for submitting titles cannot be accessed because the user has not been verified in the system. During the registration and login process, Google Recaptcha V.2 will be used as a basic security to prevent spam and other security threats.


Users who are registered as regular users will be able to see published information and submit final project and KKL titles. Meanwhile, users who are registered as admins can access the admin session, where there is a dashboard, user menu, news, final projects, KKL, file uploads, galleries, and more. Admins will manage the dashboard and provide results for the final project and KKL title submissions that have been submitted by users. Admins will provide ongoing status updates on whether the given titles are accepted or rejected.

Tags :
Laravel, PHP, HTML, Css, SEO, Bootsrap, Javascript, Responsive, Google Recaptcha, Google Search Console
